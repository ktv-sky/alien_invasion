# Alien Invasion game!

1. Install libs Pygame depends on:
    `brew install hg sdl sdl_image sdl_ttf`
2. Install Pygame
    `python3 -m pip install -U pygame --user`
3. Run alien_invasion.py
4. Enjoy!
